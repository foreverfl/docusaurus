# 고급 튜토리얼 {#advanced-tutorials}

이 섹션은 매우 체계적이지는 않지만, 다음과 같은 주제들을 다룰 것입니다:

```mdx-code-block
import DocCardList from '@theme/DocCardList';

<DocCardList />
```

여러분이 가이드를 충분히 숙지했고, 플러그인 구성 방법, React 컴포넌트 작성 방법 등과 같은 기본 사항을 알고 있다고 가정하겠습니다. 이 섹션들은 플러그인 작성자와 코드 기여자를 염두에 두고 작성되었기 때문에, [플러그인 API](../api/plugin-methods/README.mdx)나 다른 아키텍처 세부 사항을 간혹 언급할 수 있습니다. 모든 것을 이해하지 못하더라도 당황하지 마세요😉
