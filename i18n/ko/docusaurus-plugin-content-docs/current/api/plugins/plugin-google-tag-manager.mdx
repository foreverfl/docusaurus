# 📦 plugin-google-tag-manager {#plugin-google-tag-manager}

import APITable from '@site/src/components/APITable';

[Google Tag Manager (gtm.js)](https://developers.google.com/tag-platform/tag-manager)를 Docusaurus 사이트에 추가하기 위한 플러그인입니다. 사용자들이 사이트를 어떻게 사용하는지 심층 분석하려면 이 플러그인을 표준 [gtag 플러그인](./plugin-google-gtag.mdx)과 함께 사용하세요.

:::tip

[Google의 Tag Assistant](https://tagassistant.google.com/) 도구를 사용하여 태그 매니저가 올바르게 설정되었는지 확인할 수 있습니다!

:::

:::warning 프로덕션 전용

이 플러그인은 통계 데이터가 왜곡되는 것을 방지하기 위해 개발 환경에서는 항상 비활성화되어 있으며 **프로덕션 환경에서만 활성화**됩니다.

:::

## 설치 {#installation}

```bash npm2yarn
npm install --save @docusaurus/plugin-google-tag-manager
```

:::tip

`@docusaurus/preset-classic` 프리셋을 사용하는 경우 이 플러그인을 의존성으로 설치할 필요가 없습니다.

[프리셋 옵션](../../using-plugins.mdx#docusauruspreset-classic)을 통해 이 플러그인을 구성할 수 있습니다.

:::

### 구성 예시 {#ex-config}

프리셋 옵션이나 플러그인 옵션을 통해 이 플러그인을 구성할 수 있습니다.

:::tip

대부분의 Docusaurus 사용자는 프리셋 옵션을 통해 이 플러그인을 구성합니다.

:::

```js config-tabs
// Preset Options: googleTagManager
// Plugin Options: @docusaurus/plugin-google-tag-manager

const config = {
  containerId: 'GTM-12345',
};
```
