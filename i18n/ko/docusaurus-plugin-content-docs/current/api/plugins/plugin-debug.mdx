# 📦 plugin-debug {#plugin-debug}

```mdx-code-block
import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';
```

디버그 플러그인은 [`http://localhost:3000/__docusaurus/debug`](http://localhost:3000/__docusaurus/debug)에 유용한 디버그 정보를 표시합니다.

주로 플러그인 작성자에게 유용하며, 생성된 라우트 같은 `.docusaurus` 폴더의 내용을 더 쉽게 검사할 수 있을 뿐만 아니라 `contentLoaded` 생명주기를 통해 로드된 플러그인 데이터와 같이 디스크에 절대 기록되지 않는 데이터 구조도 검사할 수 있습니다.

:::info

클래식 프리셋을 통해 플러그인을 사용하는 경우, 프리셋은 기본적으로 민감한 정보가 노출되는 것을 방지하기 위해 `debug: undefined`를 통해 **개발 환경에서는 플러그인을 활성화하고 프로덕션에서는 비활성화**합니다. 항상 활성화하려면 `debug: true`를, 항상 비활성화하려면 `debug: false`를 사용할 수 있습니다.

독립형 플러그인을 사용하는 경우 환경을 확인하여 동일한 효과를 얻어야 할 수 있습니다:

```js title="docusaurus.config.js"
export default {
  plugins: [
    // highlight-next-line
    process.env.NODE_ENV === 'production' && '@docusaurus/plugin-debug',
  ].filter(Boolean),
};
```

:::

:::note

버그를 보고하는 경우, 배포 구성을 더 쉽게 검사할 수 있도록 프로덕션에서 이 플러그인을 활성화하도록 요청할 수 있습니다.

민감한 정보가 없다면 [우리처럼](/__docusaurus/debug) 프로덕션에서도 활성화 할 수 있습니다.

:::

## 설치 {#installation}

```bash npm2yarn
npm install --save @docusaurus/plugin-debug
```

:::tip

`@docusaurus/preset-classic` 프리셋을 사용하는 경우 이 플러그인을 의존성으로 설치할 필요가 없습니다.

[프리셋 옵션](../../using-plugins.mdx#docusauruspreset-classic)을 통해 이 플러그인을 구성할 수 있습니다.

:::

## 구성 {#configuration}

이 플러그인은 현재 옵션이 없습니다.

### 구성 예시 {#ex-config}

프리셋 옵션이나 플러그인 옵션을 통해 이 플러그인을 구성할 수 있습니다.

:::tip

대부분의 Docusaurus 사용자는 프리셋 옵션을 통해 이 플러그인을 구성합니다.

:::

```mdx-code-block
<Tabs groupId="api-config-ex">
<TabItem value="preset" label="프리셋 옵션">
```

프리셋을 사용하는 경우 [프리셋 옵션](../../using-plugins.mdx#docusauruspreset-classic)을 통해 이 플러그인을 구성하세요:

```js title="docusaurus.config.js"
export default {
  presets: [
    [
      '@docusaurus/preset-classic',
      {
        // highlight-next-line
        debug: true, // 프로덕션에서 플러그인을 활성화합니다
      },
    ],
  ],
};
```

```mdx-code-block
</TabItem>
<TabItem value="plugin" label="플러그인 옵션">
```

독립형 플러그인을 사용하는 경우 플러그인에 직접 옵션을 제공하세요:

```js title="docusaurus.config.js"
export default {
  // highlight-next-line
  plugins: ['@docusaurus/plugin-debug'],
};
```

```mdx-code-block
</TabItem>
</Tabs>
```
