# 📦 plugin-rsdoctor {#plugin-rsdoctor}

import APITable from '@site/src/components/APITable';

[Rsdoctor](https://rsdoctor.dev/) 플러그인은 Webpack과 Rspack 모두를 지원하며 Docusaurus 사이트의 번들링 단계 문제를 해결하는 데 도움을 줄 수 있습니다.

:::tip

어떤 플러그인이나 로더가 번들러를 늦추고 있는지 파악하고 병목 현상을 최적화하는 데 집중하기 위해 사용하세요.

:::

## 설치 {#installation}

```bash npm2yarn
npm install --save @docusaurus/plugin-rsdoctor
```

### 구성 예시 {#ex-config}

플러그인 옵션을 통해 이 플러그인을 구성할 수 있습니다.

```js title="docusaurus.config.js"
export default {
  plugins: [
    [
      'rsdoctor',
      {
        rsdoctorOptions: {
          mode: 'lite',
        },
      },
    ],
  ],
};
```
