# 📦 plugin-google-gtag {#plugin-google-gtag}

import APITable from '@site/src/components/APITable';

기본 [Global Site Tag (gtag.js)](https://developers.google.com/tag-platform/gtagjs) 플러그인입니다. 이벤트 데이터를 Google Analytics, Google Ads 및 Google Marketing Platform으로 전송할 수 있게 해주는 JavaScript 태깅 프레임워크와 API입니다. 이 섹션에서는 Google Analytics를 위한 글로벌 사이트 태그를 활성화하도록 Docusaurus 사이트를 구성하는 방법을 설명합니다.

:::tip

[Google의 Tag Assistant](https://tagassistant.google.com/) 도구를 사용하여 gtag가 올바르게 설정되었는지 확인할 수 있습니다!

:::

:::warning 프로덕션 전용

이 플러그인은 통계 데이터가 오염되는 것을 방지하기 위해 개발 환경에서는 항상 비활성화되어 있으며 **프로덕션 환경에서만 활성화**됩니다.

:::

## 설치 {#installation}

```bash npm2yarn
npm install --save @docusaurus/plugin-google-gtag
```

:::tip

`@docusaurus/preset-classic` 프리셋을 사용하는 경우 이 플러그인을 의존성으로 설치할 필요가 없습니다.

[프리셋 옵션](../../using-plugins.mdx#docusauruspreset-classic)을 통해 이 플러그인을 구성할 수 있습니다.

:::

## 구성 {#configuration}

허용되는 필드:

```mdx-code-block
<APITable>
```

| 이름 | 타입 | 기본값 | 설명 |
| --- | --- | --- | --- |
| `trackingID` | <code>string \| string[]</code> | **필수** | gtag 서비스의 추적 ID입니다. 여러 ID를 제공할 수 있습니다. |
| `anonymizeIP` | `boolean` | `false` | 요청을 보낼 때 IP를 익명화할지 여부입니다. |

### 구성 예시 {#ex-config}

프리셋 옵션이나 플러그인 옵션을 통해 이 플러그인을 구성할 수 있습니다.

:::tip

대부분의 Docusaurus 사용자는 프리셋 옵션을 통해 이 플러그인을 구성합니다.

:::

```js config-tabs
// Preset Options: gtag
// Plugin Options: @docusaurus/plugin-google-gtag

const config = {
  trackingID: 'G-999X9XX9XX',
  anonymizeIP: true,
};
```
