# 📦 theme-search-algolia {#theme-search-algolia}

이 테마는 Algolia DocSearch와 쉽게 통합되는 `@theme/SearchBar` 컴포넌트를 제공합니다. `@docusaurus/theme-classic`과 결합하면 매우 쉬운 검색 통합을 제공합니다. [검색](../../search.mdx) 문서에서 더 자세한 내용을 확인할 수 있습니다.

```bash npm2yarn
npm install --save @docusaurus/theme-search-algolia
```

이 테마는 또한 OpenSearch를 지원하는 `/search` 경로에서 사용할 수 있는 검색 페이지(스위즐 할 수 있는 `SearchPage` 컴포넌트)를 추가합니다. `themeConfig.algolia.searchPagePath`를 통해 이 기본 경로를 변경할 수 있습니다. 검색 페이지를 비활성화하려면 `false`를 사용하세요.

:::tip

`@docusaurus/preset-classic`을 설치했다면, 이것을 의존성으로 설치할 필요가 없습니다.

:::
