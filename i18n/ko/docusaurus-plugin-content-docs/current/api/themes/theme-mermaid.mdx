# 📦 theme-mermaid {#theme-mermaid}

이 테마는 [mermaid](https://mermaid-js.github.io/)로 구동되는 `@theme/Mermaid` 컴포넌트를 제공합니다. [다이어그램](../../guides/markdown-features/markdown-features-diagrams.mdx) 문서에서 더 자세한 내용을 확인할 수 있습니다.

```bash npm2yarn
npm install --save @docusaurus/theme-mermaid
```

## 구성 {#configuration}

```js title="docusaurus.config.js"
export default {
  themes: ['@docusaurus/theme-mermaid'],
  // Markdown에서 Mermaid 코드 블록이 작동하려면,
  // 이 옵션으로 Remark 플러그인도 활성화해야 합니다
  markdown: {
    mermaid: true,
  },
};
```
