# 📦 theme-classic {#theme-classic}

import APITable from '@site/src/components/APITable';

Docusaurus를 위한 클래식 테마입니다.

더 자세한 구성 정보는 [테마 구성 페이지](theme-configuration.mdx)를 참조하세요.

```bash npm2yarn
npm install --save @docusaurus/theme-classic
```

:::tip

`@docusaurus/preset-classic`을 설치했다면, 이것을 의존성으로 설치할 필요가 없습니다.

:::

## 구성 {#configuration}

허용되는 필드:

```mdx-code-block
<APITable>
```

| 옵션 | 타입 | 기본값 | 설명 |
| --- | --- | --- | --- |
| `customCss` | <code>string[] \| string</code> | `[]` | [클라이언트 모듈](../../advanced/client.mdx#client-modules)로 전역적으로 가져올 스타일시트입니다. 상대 경로는 사이트 디렉토리를 기준으로 해석됩니다. |

```mdx-code-block
</APITable>
```

:::note

테마에 대한 대부분의 구성은 [테마 구성](./theme-configuration.mdx)에서 찾을 수 있는 `themeConfig`에서 수행됩니다.

:::

### 구성 예시 {#ex-config}

프리셋 옵션이나 플러그인 옵션을 통해 이 테마를 구성할 수 있습니다.

:::tip

대부분의 Docusaurus 사용자는 프리셋 옵션을 통해 이 플러그인을 구성합니다.

:::

```js config-tabs
// Preset Options: theme
// Plugin Options: @docusaurus/theme-classic

const config = {
  customCss: './src/css/custom.css',
};
```
