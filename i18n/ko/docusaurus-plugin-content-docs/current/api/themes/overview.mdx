# Docusaurus 테마 {#docusaurus-themes}

공식 Docusaurus 테마를 제공합니다.

## 메인 테마 {#main-themes}

메인 테마는 [docs](../plugins/plugin-content-docs.mdx), [blog](../plugins/plugin-content-blog.mdx), [pages](../plugins/plugin-content-pages.mdx) 플러그인의 사용자 인터페이스를 구현합니다.

- [@docusaurus/theme-classic](./theme-classic.mdx)
- 🚧 다른 테마들도 계획되어 있음

:::warning

모든 테마가 동일한 기능, 사용자 경험 및 구성을 공유하는 것이 목표입니다.

UI 디자인과 기본 스타일링 프레임워크만 변경되어야 하며, 쉽게 테마를 변경할 수 있어야 합니다.

아직 이 단계까지는 도달하지 못했습니다. 클래식 테마만 프로덕션 준비가 되어 있습니다.

:::

## 향상 테마 {#enhancement-themes}

이러한 테마들은 추가적인 사용자 인터페이스 관련 기능으로 기존 메인 테마를 향상시킬 것입니다.

- [@docusaurus/theme-live-codeblock](./theme-live-codeblock.mdx)
- [@docusaurus/theme-search-algolia](./theme-search-algolia.mdx)
