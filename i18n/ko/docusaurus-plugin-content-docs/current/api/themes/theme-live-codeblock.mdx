# 📦 theme-live-codeblock {#theme-live-codeblock}

이 테마는 react-live로 구동되는 `@theme/CodeBlock` 컴포넌트를 제공합니다. [대화형 코드 에디터](../../guides/markdown-features/markdown-features-code-blocks.mdx#interactive-code-editor) 문서에서 더 자세한 내용을 확인할 수 있습니다.

```bash npm2yarn
npm install --save @docusaurus/theme-live-codeblock
```

### 구성 {#configuration}

```js title="docusaurus.config.js"
export default {
  plugins: ['@docusaurus/theme-live-codeblock'],
  themeConfig: {
    liveCodeBlock: {
      /**
       * 라이브 플레이그라운드의 위치, 에디터의 위 또는 아래
       * 가능한 값: "top" | "bottom"
       */
      playgroundPosition: 'bottom',
    },
  },
};
```
