# prefer-docusaurus-heading {#prefer-docusaurus-heading}

Docusaurus [`theme-classic`](../../themes/theme-classic.mdx)에서 제공하는 `@theme/Heading` 테마 컴포넌트가 헤딩에 `<hn>` 태그 대신 사용되도록 보장합니다.

## 규칙 세부사항 {#details}

이 규칙에 대한 **잘못된** 코드 예시:

```html
<h1>This is heading 1</h1>

<h2>This is heading 2</h2>

<h3>This is heading 3</h3>
```

이 규칙에 대한 **올바른** 코드 예시:

```javascript
import Heading from '@theme/Heading'

<Heading as='h1'>This is heading 1</Heading>

<Heading as='h2'>This is heading 2</Heading>

<Heading as='h3'>This is heading 3</Heading>
```
