# no-untranslated-text {#no-untranslated-text}

import APITable from '@site/src/components/APITable';

JSX의 텍스트 레이블이 번역 호출로 감싸져 있도록 강제합니다.

[i18n 기능](../../../i18n/i18n-introduction.mdx)이 사용될 때, 이 규칙은 웹사이트에 나타나는 모든 레이블이 번역 가능하도록 보장하여 어떤 문자열도 실수로 번역되지 않은 채로 남지 않도록 합니다.

## 규칙 세부사항 {#details}

이 규칙에 대한 **잘못된** 코드 예시:

```js
// Hello World가 번역되지 않음
<Component>Hello World</Component>
```

이 규칙에 대한 **올바른** 코드 예시:

```js
// Hello World가 번역됨
<Component>
  <Translate>Hello World</Translate>
</Component>
```

## 규칙 설정 {#configuration}

허용되는 필드:

```mdx-code-block
<APITable>
```

| 옵션 | 타입 | 기본값 | 설명 |
| --- | --- | --- | --- |
| `ignoredStrings` | `string[]` | `[]` | 이 목록에 있는 문자열만 포함하는 텍스트 레이블은 보고되지 않습니다. |

```mdx-code-block
</APITable>
```

## 사용하지 말아야 할 경우 {#when-not-to-use}

[i18n 기능](../../../i18n/i18n-introduction.mdx)을 사용하지 않는 경우 이 규칙을 비활성화할 수 있습니다. 또한 텍스트가 번역될 필요가 없는 경우에도 이 규칙을 비활성화할 수 있습니다.
