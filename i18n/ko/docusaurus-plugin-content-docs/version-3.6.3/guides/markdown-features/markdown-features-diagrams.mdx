# 다이어그램 {#diagrams}

다이어그램은 코드 블록에서 [Mermaid](https://mermaid-js.github.io/mermaid/)를 사용하여 렌더링할 수 있습니다.

## 설치 {#installation}

```bash npm2yarn
npm install --save @docusaurus/theme-mermaid
```

`docusaurus.config.js`에서 플러그인 `@docusaurus/theme-mermaid`를 추가하고 `markdown.mermaid`를 `true`로 설정하여 Mermaid 기능을 활성화하세요.

```js title="docusaurus.config.js"
export default {
  markdown: {
    mermaid: true,
  },
  themes: ['@docusaurus/theme-mermaid'],
};
```

## 사용법 {#usage}

`mermaid` 언어로 코드 블록을 추가하세요:

````md title="Mermaid 다이어그램 예시"
```mermaid
graph TD;
    A-->B;
    A-->C;
    B-->D;
    C-->D;
```
````

```mermaid
graph TD;
    A-->B;
    A-->C;
    B-->D;
    C-->D;
```

Mermaid 문법에 대한 자세한 내용은 [Mermaid 문법 문서](https://mermaid-js.github.io/mermaid/#/./n00b-syntaxReference)를 참조하세요.

## 테마 설정 {#theming}

다이어그램의 다크와 라이트 테마는 `docusaurus.config.js`의 `themeConfig`에서 `mermaid.theme` 값을 설정하여 변경할 수 있습니다. 라이트 모드와 다크 모드 모두에 대해 테마를 설정할 수 있습니다.

```js title="docusaurus.config.js"
export default {
  themeConfig: {
    mermaid: {
      theme: {light: 'neutral', dark: 'forest'},
    },
  },
};
```

Mermaid 다이어그램 테마에 대한 자세한 내용은 [Mermaid 테마 문서](https://mermaid-js.github.io/mermaid/#/theming)를 참조하세요.

## Mermaid 설정 {#configuration}

`mermaid.options`의 옵션들은 직접 `mermaid.initialize`에 전달됩니다:

```js title="docusaurus.config.js"
export default {
  themeConfig: {
    mermaid: {
      options: {
        maxTextSize: 50,
      },
    },
  },
};
```

사용 가능한 설정 옵션은 [Mermaid 설정 문서](https://mermaid-js.github.io/mermaid/#/./Setup?id=configuration)와 [Mermaid 설정 타입](https://github.com/mermaid-js/mermaid/blob/master/packages/mermaid/src/config.type.ts)을 참조하세요.

## 동적 Mermaid 컴포넌트 {#component}

동적 다이어그램을 생성하려면 `Mermaid` 컴포넌트를 사용할 수 있습니다:

```mdx title="동적 Mermaid 컴포넌트 예시"
import Mermaid from '@theme/Mermaid';

<Mermaid
  value={`graph TD;
    A-->B;
    A-->C;
    B-->D;
    C-->D;`}
/>
```
